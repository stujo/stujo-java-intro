# Data Types

* Understand and use [Java Data Types](http://docs.oracle.com/javase/tutorial/java/nutsandbolts/datatypes.html)

* Default Values (only for fields)
 * ``byte``	-> ``0``
 * ``short`` -> ``0``
 * ``int`` -> ``0``
 * ``long`` -> ``0L``
 * ``float`` -> ``0.0f``
 * ``double`` -> ``0.0d``
 * ``char`` -> ``'\u0000'``
 * ``String`` (or any object) -> ``null``
 * ``boolean`` -> ``false``
