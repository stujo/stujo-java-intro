# Object Basics

* Every class extends ``Object``
* ``toString`` -> type plus hashCode
* ``getClass()`` -> ``Class``
* ``instanceof``casting - A lot less of it now we have Generic Collections
* ``clone`` - ``Cloneable`` Marker interface
